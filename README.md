# Bioinformatics — Gelfand / Skoltech Coursework

This repository contains bioinformatics coursework projects completed as part of the Gelfand / Skoltech bioinformatics course. The projects cover core computational biology methods, from sequence alignment and database search to motif discovery, protein structure analysis, phylogenetics, gene prediction, NGS read processing, and genome annotation.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Pairwise Alignment](BioInf_01f_pairwise_alignment.pdf) | Introduction to pairwise sequence alignment. Covers how biological sequences are compared using scoring schemes, substitutions, insertions/deletions, and alignment quality measures. |
| 2 | [BLAST](BioInf_02_blast.pdf) | Practical work with BLAST-based sequence similarity search. Focuses on finding homologous sequences, interpreting hits, E-values, alignment scores, and biological relevance of matches. |
| 3 | [Motif Search](BioInf_03motif_search.pdf) | Search for conserved sequence motifs in biological sequences. Includes motif representation, pattern discovery, and interpretation of possible regulatory or functional sequence elements. |
| 4 | [Protein Structure](BioInf_04_protein_structure.pdf) | Analysis of protein structure and function. Covers structural databases, protein domains, secondary/tertiary structure features, and links between sequence, structure, and biological function. |
| 5 | [Phylogenetic Trees](BioInf_08_trees.pdf) | Construction and interpretation of evolutionary trees. Focuses on sequence-based phylogeny, evolutionary distance, tree topology, and biological conclusions from phylogenetic relationships. |
| 6 | [Gene Prediction in Prokaryotes](BioInf_10_gene_prediction_in_prokaryotes.pdf) | Prediction and annotation of protein-coding genes in prokaryotic genomes. Includes open reading frames, gene structure, start/stop codons, and computational genome annotation logic. |
| 7 | [NGS Raw Reads Analysis](BioInf_11_NGS_raw_reads_analysis.pdf) | Processing and quality analysis of next-generation sequencing reads. Covers raw read inspection, quality control, preprocessing, and preparation of sequencing data for downstream analysis. |
| 8 | [Final Project: Genome Fragment Annotation](final_project_genome_fragment_annotation.pdf) | Final integrative project focused on annotating an unknown genome fragment using multiple bioinformatics approaches, including sequence search, gene prediction, functional annotation, and comparative analysis. |
